# winpymenu
Python cli menu for python

# Usage
No external dependencies are required!
### Code Example:
```python
from winpymenu import WinPySimpleMenu

menu = WinPySimpleMenu(options=["option1", "option2"])

menu.show()
```

### Output:
```bash
> option1
> option2
```
